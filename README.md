This campaign is only for the A-4E-C mod of DCS World and concerns a series of 11 missions on the Kola map. This is a fictional large-scale attack by the Warsaw Pact against NATO with the aim of conquering the Scandinavian peninsula.
We are in 1961, during the "Cold War".

To play, you must have the prerequisites and each time you run the mission Αurora_Borealis_Episode.START from the DCS World main menu and the MISSION option. It is designed in such a way that it will load your progress in the campaign each time.

Prerequisites:
    • A-4E-C mod installed.
    • Persistence enabled with AstroSuite - DCS : https://github.com/astrolavos1998/AstroSuite-DCS

Alternative download:
